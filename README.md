# 🪶 Zhuge AI OS

by Mr. KM

Zhuge AI OS 不是一套功能很多的系統，而是一位會陪著使用者完成工作的 AI 工作夥伴。

第一版產品主線很單純：

```text
一句自然語言
↓
Mr. KM 理解
↓
使用者確認
↓
寫入工時月曆
↓
月底匯出 ECP
```

## Product Charter

- 每一次互動，都讓 Mr. KM 更懂你的工作。
- 每一次學習，都讓下一次工時更容易完成。
- AI 負責理解與建議，最後決定權永遠在使用者。
- 文件不是目的；文件只是讓 Mr. KM 學習工作的教材。

產品原則：[Teach Mr. KM, Not Manage Files](docs/product-principles.md)

## 目前能力

- 自然語言建立工時，確認後寫入正式工時月曆。
- Web / Mobile / Chrome Extension 共用同一份 Conversation Core。
- 工時、我的工作、ECP 設定、Conversation 與藏書閣走 Supabase Cloud Sync。
- 🪶 Mr. KM 建議以「我的工作」為來源，協助使用者更快加入工時。
- 藏書閣用來教 Mr. KM 學會新的工作，而不是管理檔案。
